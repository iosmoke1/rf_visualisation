# Random Forest Visualisation

Интерактивная визуализация Random Forest на задаче бинарной классификации.

Репозиторий: https://github.com/iosmoke1/rf_visualisation

## Что уже есть

- генерация простых датасетов: две луны, кольца, XOR, кластеры;
- пошаговое обучение леса по одному дереву;
- настройка гиперпараметров: число деревьев, глубина, `min_samples_split`, `min_samples_leaf`, bootstrap-доля, `max_features`;
- визуализация границы решений;
- просмотр первых деревьев леса;
- метрики на train/test и confusion matrix;
- важность признаков по суммарному уменьшению Gini impurity.

## Запуск локально

Проект статический, поэтому можно открыть `index.html` в браузере.

## Jupyter Notebook

Файл `random_forest_demo.ipynb` показывает обучение `sklearn.ensemble.RandomForestClassifier`, метрики качества, confusion matrix, границу решений, важность признаков и одно дерево из леса.

## Архитектура

Приложение сделано как обычная статическая страница без сервера. Браузер открывает `index.html`, подключает стили из `style.css` и весь основной код из `app.js`.

`index.html` хранит разметку: панели с настройками, область с графиком, блоки метрик и контейнеры для деревьев. В `style.css` описан внешний вид страницы и адаптация под разный размер экрана.

Основная логика находится в `app.js`. В нём генерируются датасеты, обучаются деревья, собирается лес, считаются метрики и перерисовывается интерфейс. Граница решений рисуется на `canvas`, потому что её удобно часто обновлять при изменении параметров или количества обученных деревьев.

Порядок работы:

1. пользователь выбирает датасет и параметры леса;
2. скрипт генерирует точки и делит их на train/test;
3. при обучении добавляются деревья;
4. после каждого шага обновляются граница решений, метрики, важности признаков и список деревьев.

## Связь с алгоритмом

Каждое дерево строит бинарные разбиения по признакам `x` и `y`.
Критерий выбора разбиения - максимальное уменьшение Gini impurity.
Random Forest обучает много деревьев на bootstrap-выборках и усредняет их голоса.
